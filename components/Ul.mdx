# Ul

Use Ul for unordered lists. There is not a special component for `li`, simply use a plain `li` html tag.

## Examples

<Playground>
  <Text>Today we learn about three types of fruit.</Text>
  <Ul>
    <li>Apples</li>
    <li>Bananas</li>
    <li>Oranges</li>
  </Ul>
  <Text>Eat more fruit.</Text>
</Playground>
